Circuits: Example

Input: Addend bits + carry-in bit

Output: Sum bits + carry-out bit

Function: Compute sum using and, xor, and or gates

Timing: Depends on the slowest path. Assuming that propagation delay is 18ns for and, 22ns for or, and 30ns for xor, the total time delay for the circuit is 70 ns.

Circuit Types

Trigger Mechanisms: Level-triggered

Stores new signal only when clock signal is in HIGH (LOW) state, maintains current value otherwise

Sequential Logic

Trigger Mechanisms: Edge-triggered

Stores new signal only when clock signal transitions from HIGH to LOW (LOW to HIGH) state, maintains current value otherwise

D latch is high level-triggered while D flip-flop is positive edge-triggered

Sequential Logic

Register

N number of flip-flops (sharing a common clock) form an n-bit register!

Sequential Logic

Register Files

A register file is a collection of registers!

Specific registers are addressed via Read/Write Register Ports

Data stored in registers are read/overwritten via Read/Write Data Ports

Sequential Logic

Register Files: Reading

Reading process done on the same clockcycle

Width of read register port depends on the number of registers contained in the register file (32 registers require 5-bit width)

Width of the read data port depends on the word size of the processor/register file

Sequential Logic

Register Files: Writing

Writing process is not immediate

Width of write register port depends on the number of registers contained in the register file (32 registers require 5-bit width)

Width of the write data port depends on the word size of the processor/register file
