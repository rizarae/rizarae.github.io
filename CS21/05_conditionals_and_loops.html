When the MIPS processor loads data from memory it consults a register regarding the location. How does it know where to fetch an instruction?

Program Counter

Special register which contains the address of the next instruction that would be fetched from memory

Branch Instructions

Branch Instruction

Instructions that change the value of the Program Counter

Conditional Branch Instructions

Branch instructions that change the value of the Program Counter only when certain conditions are met

Branch if less than (blt), branch if greater than (bgt)
