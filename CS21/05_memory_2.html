Main Memory

Main Memory

  1. Memory that is between the cache and the external memory
  2. Outside the CPU, but still in the mainboard/motherboard
  3. Usually comes in modular packages
  4. Implemented using semiconductor technology

External Memory

  • Lowest in the memory hierarchy
  • Largest in capacity, slowest in speed
  • Most common are magnetic and optical disks

Magnetic Disks
