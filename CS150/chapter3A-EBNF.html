Extended BNF

[ ]

Optional parts are placed in brackets

<if_stmt> → if (expression) <statement> [else <statement>]

( | )

Alternative items are placed inside a set of parentheses and are separated by a vertical bar

<term> → <term> (* | / | %) <factor>

{ }

Repetitions of 0 or more are placed inside braces

<term> → <factor> {(* | / | %) <factor>}
